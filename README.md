# Smart City Weather Dashboard

## Project Description
The Smart City Weather Dashboard is a web application that provides real-time weather information for any city using the OpenWeatherMap API. The application also provides clothing suggestions based on the weather conditions and temperature.

## Purpose
The purpose of this project is to help users quickly check weather conditions and get suggestions on what to wear based on the weather forecast.

## Features Planned
- Search weather by city
- Get current weather data
- Get 5-day weather forecast
- Clothing suggestions based on temperature and weather
- Dynamic weather icons
- Background color change based on temperature
- Auto-detect user location using Geolocation API
- Filter forecast by weather condition (Rain, Clear, Clouds)
- Sort forecast by temperature

## API Used
OpenWeatherMap API  
https://openweathermap.org/api

## Technologies Used
- HTML
- CSS
- JavaScript
- OpenWeatherMap API
- Geolocation API
- GitHub

## How the Project Will Work (Plan)
1. User enters city name
2. App fetches weather data from API
3. App displays current weather and forecast
4. App gives clothing suggestions
5. Background color changes based on temperature
6. User can filter and sort forecast data

## Setup Instructions
1. Clone the GitHub repository
2. Open the project folder
3. Create a file named config.js
4. Add your OpenWeatherMap API key:
   const API_KEY = "your_api_key_here";
5. Open index.html in your browser
